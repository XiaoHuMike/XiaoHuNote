# 在 C# 中，类型转换有多种写法，包括单个对象和集合对象的转换。下面是常见的类型转换写法：

  1. 单个对象的类型转换：

    • 使用强制类型转换（Casting）：使用括号将目标类型括起来，将要转换的对象放在括号前面。

      object obj = "Hello";
string str = (string)obj;
      1
      2

    • 使用 as 运算符：将对象转换为目标类型，如果转换失败则返回 null。

      object obj = "Hello";
string str = obj as string;
      1
      2

    • 使用 Convert 类：使用 Convert.ToXxx 方法将对象转换为目标类型，例如 Convert.ToInt32、Convert.ToDouble 等。

      object obj = "123";
int num = Convert.ToInt32(obj);
      1
      2

    • CastTo 和括号一样，只是不报错，返回true和false

  2. 集合对象的类型转换：

    • 使用 LINQ 的 Cast 方法：将集合中的元素转换为指定类型，如果存在不可转换的元素会抛出异常。

      ArrayList list = new ArrayList { 1, 2, 3 };
List<int> numbers = list.Cast<int>().ToList();
      1
      2

    • 使用 LINQ 的 OfType 方法：筛选出集合中指定类型的元素，并返回一个新的序列。

      ArrayList list = new ArrayList { 1, "two", 3 };
List<int> numbers = list.OfType<int>().ToList();
      1
      2

注意：在进行类型转换时，需要确保转换是安全的，否则可能会引发运行时异常。在某些情况下，可以使用 is 运算符来检查对象是否可以转换为特定类型，然后再进行转换。
